# Estimasi Project GEM

Struktur repo:

- `frontend/` : React + Vite + Tailwind (UI)
- `backend/`  : Node.js (Express) + Supabase Postgres (API)

## Jalankan Frontend

```bash
cd frontend
npm install
npm run dev
```

## Jalankan Backend

```bash
cd backend
npm install
npm run dev
```

## Jalankan Keduanya Sekaligus

```bash
npm run setup
npm run dev
```

## Supabase Schema

Jalankan SQL ini di Supabase SQL Editor: `backend/supabase/schema.sql`.
